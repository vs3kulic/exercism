## Usage
1. Clone repository:
```
git clone https://github.com/vs3kulic/exercism.git
```

2. Navigate to specific exercise:
```
cd exercism/<exercise-directory>
```

3. Run tests (requires pytest):
```
pytest <exercise-name>_test.py
```
